# issue-tracker

This simple Issue Tracker website using Node js, Express js, MongoDb Database and EJS Template Engine. UI for this is Responsive and user-friendly.

## Features

- Create Project
- Display Project Details.
- Create Issue For Particular Project.
- Show All Issues.
- Filter Issue using Label or Author.
- Search Issue using Issue Title or Description.


## Getting Started

Follow these steps to get started with the alarm clock code:

1. **Clone the Repository**: Clone this repository to your local machine using the following command:
    git clone https://github.com/dhanashrirandive06/issue-tracker.git
2. **Navigate to the Project Directory**: Move to the project directory:
    cd issue-tracker
3. **Run the Code**
